时间过得飞快。转眼间就九月十九号了。

今天最后一节课是物理。杨静依早早把正课讲完，趁着下课还有五分钟就丢出了一个重磅消息。

杨静依: 那个，我说一下，啊，这周四到周五学校要进行九月月考。

教室里顿时炸开了锅。

杨静依把手往讲台上一拍——啪！全班再次安静下来。

杨静依: 安静！没听到吗？本次月考考试呢，主要是考察你这一个月以来有没有认真学习，有没有按照老师的要求把你该做的事做好。
杨静依: 如果你上一个月呢，都玩去了，那，这个就不好说了。趁着还有一周，赶紧补一下。
杨静依: 不然的话，啊，考出那点分，你自己看看你自己看不看得下去。

// ============================================================
// 3-2 作业风波
// ============================================================

@bg(/Assets/backgrounds/scenes/Classroom2.jpg)
@music(/Assets/music/DayAfterDay2.mp3)

今天的作业也开始多起来了。以前是写到手酸，现在脑袋也开始疼了。好不容易做完其他科，还有一科生物，有八页练习册。
晚四下课还有十分钟了，旁边的同学都基本上抄作业抄得差不多了，我还有一半没写完。我一看表，知道只有十分钟了，顿时脑瓜子涨得疼。

但是还没完。课代表站了起来。

沈佳蕙: 收生物作业。

我的大脑一片空白。怎么办，怎么办？

收我们这组的生物课代表是沈佳蕙。此刻，她正在我前方几排的位置收一位女同学的作业。

女同学A: 我就差一道题。马上写完。

沈佳蕙撇撇嘴。

沈佳蕙: 洛老师说，没做完的当没做处理。

然后就往一个牛皮包着的小本本上记名字。她又问旁边另一位女同学。

沈佳蕙: 你呢？

女同学B: 写了。

那女同学把练习册交给她。沈佳蕙接过来，翻了两下。

沈佳蕙: 你这个写是写完了，但是你没有批改啊。

女同学B: 啊？批改？批改什么？

沈佳蕙: 作业啊。答案不是专门发了吗？洛老师说一定要对照答案认真批改，不能只改答案，要红笔圈画错误原因。这可是她上课亲自说的。看，你的作业也不符合要求。按照洛老师的要求，当没做处理。

沈佳蕙说着又打开小本本记下来。

女同学B: 烦人。

当然，我这边也没有闲着。就在沈佳蕙记名字的时候，我就已经抄完了，现在正拿着红笔对着答案批改。
但是，还没改完，就闻到一股极其浓郁的香水味道。我一瞥，沈佳蕙已经打开小本本记我名字了。

沈佳蕙: 没改完，当没做处理。

我: 逆天。

沈佳蕙: 什么？你说什么？

全班人都看向我，就连代杨静依守自习的老师也抬起头瞥了我一眼，然后就都埋头干自己的事情了，似乎刚才啥都没发生。

@ch(/Assets/characters/LinZixin/LinZixin_confused.png)

林子欣赶紧用手肘推了推我，暗示我别说了。

我也没准备开腔，沈佳蕙就双手颤抖着拿着小本本记录。

沈佳蕙: 不仅没做，而且态度不好，严重记录。

说完就去检查下一个了。

// --- 朋友前来安慰 ---

漆诚一下课就搭上了我的肩膀。

漆诚: 哥们儿，你这属实逆天了。

我苦笑了一下。

何肖也过来，虽然没像漆诚那样直接，但他也苦笑着对我摇了摇头。

何肖: 没事的哥们，管他的。有些人是这样的。

我: 明天要是叫洛锐知道了，我就寄了。

// 注释："寄"是网络用语，是"死"的一种非正式说法，此处为玩笑。

肖逸言也过来了：你跟她解释，洛锐应该会听的。

回到寝室，遇到刚从球场打完球回来大汗淋漓的张哲健。

张哲健: 哟，哥们，被沈大妈单杀了？

我见他像看乐子，就冷冰冰地嗯了一句，转身就走。

张哲健赶忙一把抓住我：别这样。沈佳蕙一直都这样的，其实。

我: 你怎么知道。

张哲健: 她是我女朋友嘛。她有时候就是这样，情绪很不稳定。你也别往心里去。

我: 哦。

// 3-3 洛锐的惩罚
// ============================================================

@bg(/Assets/backgrounds/scenes/Classroom3.jpg)
@music(/Assets/music/What.mp3)

第二天，下午第二节是生物。洛锐板着脸，气冲冲地走进教室，啪一声把包往讲台上一甩。

洛锐: ……我们有些同学，作业敷衍，写完不批改，这哪是对自己学习的这么一个良好的态度？我之前已经明确、反复强调，没做完、没批改，当没做处理。
洛锐: 还有某些同学，居然敢冲课代表发脾气，胆子大嘞！已经胆大包天了！这还了得！

全班开始有吱吱的笑声。洛锐眼睛瞪得老大。

洛锐: 昨天没认真完成作业的同学，我不点你名，自己站到教室后边去。

昨天晚上那两个同学，还有我，自觉站了出去。

洛锐指了指我。

洛锐: 你，站到教室外面去。

教室外面就教室外面呗。我叹了口气，打开后门站了出去。关上门的时候，教室里有一些"嘻嘻"的笑声。

@bg(/Assets/backgrounds/scenes/TeachingBuilding.jpg)

反正这节课我听不下去了。我望着教学楼外边的树木出神。绿色叶子摇曳着，在阴天苍白的阳光中舞动。
但我一点兴致也没有。拿着的生物书，就像突然重了好几倍。昏昏沉沉的。

打了下课铃，我就准备回教室了。

洛锐赶紧出来吼了一声。

洛锐: 不准进去！跟我去找你们杨老师。

// ============================================================
// 3-4 办公室审判
// ============================================================

@bg(/Assets/backgrounds/scenes/BuildingInside1.jpg)
@music(/Assets/music/Inside.mp3)

洛锐在前面提着那个精致的女士小包，抱着一摞生物书，以急促的小碎步进了杨静依的办公室。而我在后面亦步亦趋。

教师办公室开着十六度的空调，凉飕飕的。苍白色灯光像针扎一样刺进我的眼睛。
杨静依坐在办公桌前，桌上摆着十几个摆件，那些大眼睛的玩具娃娃本来都笑着闹着，但在这一刻，我只感到很难受。

洛锐: 杨老师，你可要为我做主啊……

洛锐进门就带着哭腔喊道。

但是这个时候，杨静依正在训着石晴雯。

杨静依: 说，你这个日记是怎么回事。

石晴雯: 我没有……我就是下课记了个日记……然后你就抢过去了……

杨静依: 记日记？那为什么要把学校名字写成"烂山中学"？

石晴雯: 一个"火"字，一个"兰"字，可不就是"烂"嘛，你自己看错了……

杨静依: 首先，石晴雯，你不要以为你是女生我就不能把你怎么样了。你自己看看，把学校称为"烂山中学"，我作为一个火兰山中学的老师，看到这个，我是很寒心的。

石晴雯: 对不起……对不起……

石晴雯已经哭了。

杨静依瞥了她一眼。

杨静依: 好吧，你回去吧。下不为例。

石晴雯哭着向杨静依鞠了一躬，转身，看了我一眼。那眼神里带着一种说不清的复杂——像是同情，又像是无奈。
她抿了抿嘴，眼眶还是红的，想说什么，但最后只是轻轻摇了摇头，走开了。

这时，杨静依才看到门口已经等待多时的洛锐。

杨静依: 诶，洛姐，麻烦你来一趟。有何贵干？

洛锐转过头来，气冲冲地指着我，好像有天大的委屈。

洛锐: 这个学生，我没办法教了！不写我的作业，还对我的课代表恶语相向！
洛锐: 杨老师，你是年轻人，是明白人，你看看，像这样性质恶劣的学生有法教吗？根本就不把老师和规则放在眼里！
洛锐: 像这样的学生，我没办法教了，教出来也是个无耻之人！

杨静依盯着我看了一会，又对洛锐说。

杨静依: 洛姐，你先坐。

洛锐就把那个精致的女士小包打开，扯了一张塑料袋垫在旁边的椅子上，好半天才坐下去，然后翘起二郎腿盯着杨静依。

杨静依高傲地盯了我一眼。

杨静依: 你，有点厉害呢。怎么回事。说。

我: 就是在昨天晚上……

杨静依扯了一张A4纸，抽了一支笔丢给我。

杨静依: 写。写你是谁，事发时干了什么，后果怎么样，有什么感想。

然后我就这么写了。我把昨晚的事完整陈述了一遍，然后我想，还是息事宁人比较好，我不想把事情闹大。
于是我写了我让课代表很生气，辜负了学科老师的期望，造成了不良影响，等等等等。

我交给杨静依。杨静依看完，扔到一边，轻蔑地说。

杨静依: 看来你还知道。

她又朝向洛锐。

杨静依: 洛姐，你看一下。

洛锐走近来，看了一眼那张纸。

洛锐: 杨老师，既然这样了，我提议，给他记一个大过处分，回家反省一个月，再来谈其他的事情。

杨静依: 大过处分和回家反省都是要先向学校教务处申报的，这我也没办法，得叫王主任来。

洛锐: 那我跟你一起去教务处说，必须要给他一点教训。

杨静依: 你别急，马上我就去给教务处王主任打电话。

我在旁边，好似晴天霹雳。这对我来说可不是什么好事。
记大过，要是记入学生个人档案就是一辈子的污点；而回家反省，我怎么能够忍受父母的痛苦和责骂。

办公室里有个老师要走了，看到我，对我语重心长地说：小伙子，同班主任和老师扳手腕，划不着呀！说完他就走了。

洛锐指着我催促说：你，站在旁边。没点规矩怎么行呢？

我依言。

杨静依: 别站那。我不喜欢有人盯着我。转过去，面对墙壁。

我依言。

望着墙壁，听见上课铃已经打了，而我还在办公室接受我应有的惩罚。

// --- 杨静依打电话 ---

杨静依: 喂，王主任，今天您有空吧？……哦，是这样的，我这里有一个学生要处理。作业不写，还脾气暴躁，不听教，不尊重老师和课代表……嗯，好的，谢谢～～麻烦了。拜拜～～

我一直面对着墙壁站着。那些先前走出去的老师，可能是听到办公室里没动静了，才陆陆续续地走进门来。我只听到哗啦哗啦抽椅子坐的声音。

杨静依的声音变得轻松起来，和其他老师聊起了天。

杨静依: 赖老师诶，你今天还有没有课啊。

赖老师: 有的咯，两个班，三节课，再累也得上啊。学生又不听……

杨静依: 没得事，反正今天办完事就下班了。回去耍！

赖老师: 诶，年轻人嘛，趁年轻，多做点嘛。

杨静依: 比不上你们天天自在，赖老师！资历又高，工资又高，天天还这么悠闲。你看，你们老一辈的，有的是方法和经验，今后有啥事还得仗着您……

赖老师: 别奉承我了，小杨嘞。工作你先做嘛。你是我最看重的，不会让你吃亏。

洛锐插了一句嘴。

洛锐: 现在教学方法好落伍哟，我昨天刷快音看到他们帛日中学，都给学生发平板电脑辅助教学了。

杨静依: 人家是人家。要是可以，谁会在这破学校呆啊。

另外一位女教师的声音：哎，就是就是。没办法，谁叫火兰山经费太低了。当年火兰山和帛中都要招我，要不是当初考进来的时候那个王者超答应除了五险一金还有年终奖，我直接去帛日中学算了。
另外一位女教师的声音：哪知道来这里这么不爽，我真是受够了。

我就一直面对着墙站着。

杨静依轻蔑地笑道：反抗有屁用，还不是活成了自己最讨厌的样子。

洛锐: 这就是命，还不如多服软少吃亏。

// ============================================================
// 3-5 王者超出现
// ============================================================

@music(/Assets/music/Warning.mp3)

就在这时，一个陌生的声音传来。

王者超: 这谁呀，啊？站在这干嘛？

我回头一看，办公室里来了一位矮胖的中年男子，一副近视眼镜戴在额头上，西装革履，腰间的皮带也压不住他的啤酒肚。

杨静依和洛锐赶紧站了起来。

洛锐: 诶，王主任，难得来一趟。

王者超: 小杨，这是不是就是你说要处理的那个学生？

杨静依: 对，就是。正考虑给他记个过，警告一下。

王者超: 他干什么了？

洛锐: 让他自己说！不做我的作业也就罢了，还公然侮辱我的课代表！侮辱我的课代表，就是侮辱我，懂吗？还不听教，这样的学生我没办法教了！

王者超: 具体原因是什么，我得知道啊。

杨静依赶紧掏出手机来给我家长发短信。

王者超眯起眼睛读完了我写的"事情经过"。

王者超: 看起来证据确凿。现在，小杨，你就把他家长叫来，我们一起处理这个事。多一个人见证也好。

杨静依: 警告吧，按照校规是这么来的。

洛锐: 不行！至少要给他记一个留校察看！王主任，你是明白人，应该知道这个事情的严重性！要是所有人都像他那样，那还得了！基本的尊重都没有！

王者超: 但是这个问题，留校察看是不是太小题大做了，洛老师。

洛锐: 不行，必须留校察看，给他长点教训！

王者超: 但是根据学校相关规定，给他的处分确实是要比你说的轻得多，大多你口头批评一下就行了。

洛锐: 我不管。这种行为极其恶劣，要是一直这样下去，那就是放任自流了。

洛锐又转向我：知道吗？这就是你们说的"犯错都是小事情"，但这个错误就是不能犯。明白吗？

洛锐接着说：哎，现在教师地位真卑微啊，不能打学生，不能骂学生。一打一骂，学生就去做自由落体运动了，最后老师上了热搜被开除。
洛锐接着说：那只能给他们纪律处分了，记在档案里影响他们一生了。等他们知道没有哪个学校、没有哪个公司愿意要他们的时候，他们就会知道错了。
洛锐接着说：没有什么更仁慈的惩罚方式了。

@wait(1.0)

王者超考虑了一下。

王者超: 要不这样，给他一个警告，这件事就了了。

洛锐: 不行！必须留校察看。

王者超: 留校察看不至于。最近我那好多早恋的，私藏电子产品的，我都记的是严重警告。

洛锐: 不行，必须留校察看！

王者超: 那就记大过吧。

洛锐: 不行！必须留校察看！

洛锐吼的嗓子都要嘶了。

王者超: 记大过已经是他这个错误的惩罚底线了，不能再高了。你是不是还想跟学校教务处扳扳手腕，洛老师？

洛锐不吭声了，头转到一边，好像还嘟囔了一句"明明之前都是我说了算"。

王者超: 那好，记大过，就这么定了。

说完，王者超打开办公室公用电脑，开始打字。

// --- 通报书 ---

违纪通报。

……同学系高2022级27班学生。经查，该同学作业完成拖沓，规定时候不交作业，且对待课代表态度非常不好，性质极其恶劣。
根据教育部《中小学生守则》和《帛日火兰山中学学生管理条例》，为了教育本人，警示他人，经学校教务处开会研究决定，现给予该高2022级27班同学以记大过处分。
特此通报。帛日火兰山中学。2022年9月20日。

@wait(2.0)

纸张刚打印出来没多久，我妈就气冲冲地来了。看着我站在那，抡起一旁的板凳就往我这砸。
洛锐虚拦了一下，赶紧把凳子抓起放在地上，那样子好像在说：这凳子可不是你来砸的。砸烂了多可惜呀。

杨静依满脸堆笑：您先消气。令公子不过是一时思想转变不过来。思想转变过来了，就好了。

妈: 这种逆子养他干嘛！养条狗都要冲着我叫几声，他倒好，天天惹事，这不，又找了你老师们的麻烦……

洛锐不屑地说：我们麻烦不麻烦，关键是要让你儿子知道他做错了。可惜，他一直都认为自己是对的，就只能烦请学校王主任代为教育了。

妈: 正事不干，天天惹事！

妈最后甩下一句，坐在凳子上。

王者超把打印纸递过来。

王者超: 事已至此，只能先给个处分了。来，签个字，盖个手印。

我做梦都想不到我能以这样的方式重大违纪。从小，父母就教导我要听话，不要跟别人对着干，不要惹事找麻烦……

可现在，闹了这么大个事情，一时真的难以接受。

一下子，我的视野就模糊了，一滴什么东西掉在了打印纸上。

洛锐幸灾乐祸地说：你现在终于知道事情的严重性了吧？

杨静依: 签字！别愣着！

王者超: 娃儿，你早知如此，又何必当初呢？当初你要是不违纪，好好听老师的话，就没有今天这个事情了。

杨静依: 就是就是，再叫你不尊重老师和课代表！

王者超: 赶紧签啊，我要下班了。

妈: 签啊！

我当时浑浑噩噩的，然后迷迷糊糊的签了字，盖了手印。

洛锐: 我建议再加一条："本人保证以后再无任何违纪。事无大小，若有违反，听从学校开除学籍处置。本人毫无怨言，绝不反悔"。
洛锐: 给他把条件焊死，叫那些学生看看什么叫"杀鸡儆猴"。今天处置一个，以后就能一劳永逸了。

妈: 写啊！

迫于无奈，我写了。洛锐和杨静依这才满意地笑起来。

@wait(2.0)

我就这么被记了大过，在那天下午的例行讲话中被点名道姓地全校通报了。

// 注释：在中国高中，"记大过"是仅次于"留校察看"和"开除学籍"的严重处分，会计入学生个人档案。此处处分与所犯错误的比例严重失衡，属于权力滥用。

// ============================================================
// 3-6 沉默的午后
// ============================================================

@bg(/Assets/backgrounds/scenes/Classroom4.jpg)
@music(/Assets/music/MySecret.mp3)

@ch(/Assets/characters/LinZixin/LinZixin_sad.png)

我跟我同桌林子欣那一整天都没说话。她也好像刻意在避开我，每次我想开口说点什么，她就低下头去翻书，或者假装在找什么东西。我心里堵得慌，但也不知道该说什么。

下课了，何肖和漆诚到我位置上找我。何肖拍了拍我的肩膀，刚想说点什么，我已经站起身，绕过椅子，头也不回地走出了教室。

走廊里人来人往，三三两两的同学聚在一起聊天、打闹。可我一走过去，声音就小了下去。有人抬头看了我一眼，很快又移开视线。
那种感觉很奇怪——不是被孤立，也不是被嘲讽，更像是……你身上突然多了一个啥，所有人都看见了，但谁也不愿意去碰它。

@bg(/Assets/backgrounds/scenes/Path.jpg)

我低着头快步走过，耳边的嘈杂声像是隔了一层玻璃。直到拐进厕所，关上门，才长出了一口气。

@wait(1.5)

@bg(/Assets/backgrounds/scenes/RoomNight.jpg)

晚上回到寝室，我躲在门后面听他们说什么。

张哲健的声音从门缝里传出来：……他这样子绝对是洛锐搞的鬼。了事化小，小事化大！

李游的声音：啥人啊！生物教的不咋样也就算了，这不纯粹把人往抑郁症整嘛！

漆诚的声音：受不了她了，一天板着个死人脸，一副看谁都不爽的样子。

何肖的声音：就没人站出来说他不应该接受这个处分吗？

李游: 你们怎么知道是洛锐要给他整这么个莫须有的处分？

// 注释："莫须有"指凭空捏造、毫无根据的罪名或指责，常用来形容诬陷或强加的不实之词。源自南宋秦桧诬陷岳飞的典故。

肖逸言: 非她莫属。洛锐战绩可查。去看论坛吧，因为类似事件，她都要被别人骂死了，差点还被开户。你别说，我还真佩服她的抗骂能力。

// 注释："开户"为网络用语，指曝光个人隐私信息（如真实姓名、住址、电话等）。

何肖最后说：草，自己制定的规矩有问题，还不允许任何人违反！我们学九科，就你生物最重要吗？

张哲健又打开了话匣子：洛锐在她管的班也是相当逆天。早上七点十分到班，中午两点十分到班，晚自习六点二十到班；大课间就十分钟，在学校的标准下砍了一半；平均吃饭时间三十分钟；生物课还要占用课间五分钟拿来背书……
依照咱们食堂这效率，吃饭可能就只有八九分钟。上课，洛锐还要不定期进教室监督，但凡上课走一点神都得完犊子。
而且更牛批的是他们的班规一共二十一条——不论男女，发烧三十九度五以上才准请假，不超过三十九度五就去她那里吃退烧药；如果你违了纪，就让你回家反省，而且立马准假！这哪是班规，要我说这特么就是"二十一条"！

// 注释："二十一条"是1915年日本向袁世凯政府提出的不平等条约，此处为夸张的调侃。

何肖: 啊？咱是来上学的还是来当机器的？哎，相比之下，我们老杨还是太保守了，起码还能按照学校本来规定的来！

肖逸言: 所以论坛上就老是说洛锐有点子毛病。

还有那个杨静依也是，刚大学毕业就来教书，脱离了学生身份，摇身一变变成老师了，了不得了。漆诚冷笑着说：她是纯为了报复而当老师的吧，顺带接受一点学校爆的金币。

我推门走了进去。所有人都看着我。过了好久，他们才上前。

漆诚: 哥们，我们追认了你为烈士！

肖逸言: 感谢成为洛锐的实验品。你放心，绝对还有后来人的。

何肖: 你为我们所有人提供了一个未曾设想的道路。

李游: 没事兄弟，大学和企事业单位不会因为你没按时交作业就不要你了。招生办老师、公司HR为什么不要你？"因为你高中作业没写完给记了个大过"，那才扯淡！

张哲健拍了拍我的肩膀。

张哲健: 下次他们再这样，给大哥我说，咱去学校教务处告他一炮！根本不虚的！

虽然他们都只是开玩笑，但是我还是挺感动的。

@wait(1.5)

// 3-7 一块巧克力
// ============================================================

@bg(/Assets/backgrounds/scenes/Classroom.jpg)
@music(/Assets/music/Noon.mp3)

第二天，我在桌子上的练习册底下发现了一块学校小卖部卖的巧克力。我正在拿起来观察思考是谁丢在这的时候，旁边的林子欣瞄了我一眼，笑了笑。

@ch(/Assets/characters/LinZixin/LinZixin_smile.png)

林子欣: 给你的。甜的东西会让你开心哦。我就是甜口，所以我天天开开心心的。不要太在意那些了。

// 注释："甜口"指爱好甜味的人。甜味会刺激大脑皮层相应区域产生积极的心理反应。

@wait(1.0)

// ============================================================
// 3-8 月考降临
// ============================================================

@music(/Assets/music/DayAfterDay.mp3)

杨静依走进教室，站在讲台上。

杨静依: 这周星期五，是你们进校的第一次月考。我希望各位，拿出你们真正的水平，不要作弊，诚信考试。这一次考试的成绩我会公布到家长群里面，家校联动，那肯定也要发挥家长的作用啊。

我抓了抓脑袋，看向林子欣。

我: 你觉得你有没有把握？

@ch(/Assets/characters/LinZixin/LinZixin_normal.png)

林子欣: 没有，我感觉这次会考的很差，我也什么都没有复习到。

生物课上，洛锐依旧是那样的疯疯癫癫。

洛锐: 这次月考的生物命题就是我负责的。我要让你们知道两件事情：第一，生物是理科不是文科；第二，你们看到命题人是我的时候，这张卷子就没有必要抱着及格的希望去写了。

林子欣小声对我说：不见得，我看洛锐这人虽然平时疯疯癫癫的，但实力这一块我还是很畏惧的。

终于，高中第一次月考来临了。

第一天考的是三门理科。生物正如洛锐说的那样，全是情景题和创新题，有些地方无从下笔。物理化学都还好——不会的还是不会。
第二天是三大主科。数学正如李大泉说的那样，前面倒是十分简单，但写完前两道选择题后面就开始连蒙带猜了。
剩下的文科嘛，没怎么背，瞎写一通，一气呵成。

// ============================================================
// 3-9 成绩公布
// ============================================================

@bg(/Assets/backgrounds/scenes/Classroom1.jpg)

就这样浑浑噩噩地过了几天，评讲卷子，上新课。很快就到了出成绩的日子。

@ch(/Assets/characters/LinZixin/LinZixin_normal.png)

林子欣: 怎么了？怎么一脸不高兴的样子？

我: 感觉考差了，对答案的时候都错了好多道题。

林子欣: 哎，怕什么，我物理还连错了四道选择题呢。听他们说何肖这次考得才好，物理选择题全对。

杨静依走了进来。

杨静依: 好了好了，安静。现在开始念成绩啊。哎哟，你们这次考得才差。你们是我带过的最差的一届！

何肖嘟囔了一句：你不才当老师吗？引得全班哄堂大笑。

杨静依: 哟，何肖，你胆肥了是吧，啊？我看看你成绩啊。你也就物理考好了。物理也才九十七分——单科全班第一、年级第六。总分班级排名第六。要不是我天天说着你，你的物理能考这么高？好了，我来拉通念一下吧。

全班的情绪紧张到了极点。

杨静依特意清了清嗓子。

杨静依: 第一名，石晴雯……第三名，林子欣，数学考得可以，一百三十八分，单科全班第一。

// 注释：高中数学满分150分，138分已是顶尖水平。

陈辛芷转过头：林姐，你不是说你考得差吗？

江诗轩也微笑着看着她：是啊，不是才第三名吗？

林子欣无奈地摆摆手，脸上却笑开了花。

杨静依继续念：第九名，肖逸言。生物可以，九十八分，全年级第三，也是我们班第一。

我一直听着消息，生怕错过自己。

第十三名，漆诚。第十七名，江诗轩。第二十五名，马武。第三十二名，张哲健……

我在心里急切地恳求：老天爷，你给我一个中间的排名就可以了，求你了……

虽然内心演绎过无数遍考差的样子，但实际听到的时候还是难以接受。

第五十五名。倒数第十。

五雷轰顶。

@wait(2.0)

@ch(/Assets/characters/LinZixin/LinZixin_sad.png)

林子欣: 没事的，一次月考而已。我之前也考差过，后面努力追上来就行了。

我摆摆手，示意让林子欣不要管我。

杨静依: 这次成绩，啊，希望同学们，理性看待。成绩呢，已经发到家长群了。现在你们要做的，就是反思，冷静地反思错题，及时查漏补缺。很多人都考得不好。有些人啊，浮躁。看看自己是哪根草。

// 3-10 贫困生风波
// ============================================================

杨静依: 好了好了，你们都给我消停一下。那个，顺带说个事啊。我们学校最近有个贫困生评选的会议。
杨静依: 为了防止拉票和舞弊，学校决定采取不记名、不公开的形式投票。你们啊，待会拿到那个票就在上面打勾就行，别给我搞其他有的没的。

拿到名单，我粗略地瞟了一眼。反正都考差了，已经没心情管这些了。随缘吧，哪个名字好听就选谁吧。

这时，一个名字吸引了我的注意。

沈佳蕙？她怎么在上面？

仔细一想，沈佳蕙可说不上穷，反而还算我们这有钱的一批了。听别人说，她之前还给她的爱豆一口气花了小几千。这怎么就申请上贫困生了？

唉，算了算了，可能家家有本难念的经吧。我也没多想，就随便选了个名字勾了完事。

@wait(1.0)

几节课后，物理课下课。龙杨招呼旁边的几个男生出去。

龙杨: 诶诶，你们知道吗？那个贫困生补助有猫腻。

旁边几个男生露出不可置信的表情：什么？

龙杨把自己在办公室的经历讲了一遍——他听到老师们在谈论一个女生送挂件来换取贫困生补助。

龙杨: 我看啊，这样可不行。我有一计——我可是辩论社的人，要不我们就趁着这个机会举办一场辩论？辩题就是"究竟什么样的同学才能得到补助"。

有些男生点了点头，也有人反对：这些事情还是别闹大，我们做好自己的事情就好了。

龙杨: 不行不行，这种事情肯定要重拳出击啊！那好，既然大家都同意了，那就定在明天下午体育课吧。

@bg(/Assets/backgrounds/scenes/RoomNight.jpg)

考砸了。李游中午一进寝室门就嚷嚷。

何肖: 慌啥，兄弟，好好想想，今天九月二十六号，星期一，周末就是十月一号，国庆节啊！七天的大假，想想都兴奋！

肖逸言: 不是只有五天吗？平时国庆七天假期都是因为国庆放五天假，剩下两天假期是中秋节。但是今年的中秋节已经在九月十号教师节那天过了，所以今年国庆应该只放五天。

何肖: 唉，草，恼火。别让我知道是谁发明了调休。
